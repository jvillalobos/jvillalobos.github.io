Unlisted Extension Test

This a github.io test page for unlisted extensions. From here you can install a couple of unlisted versions of the AMO Admin Assistant extension, and test automatic updates from one version to another.

  • Version 4.2
  • Version 4.3

How to do this

  1. Set up GitHub Pages.
  2. Add a landing page (this page, maybe less crappy-looking) and an update manifest. Example.
  3. Create your unlisted versions, making sure you include an update_url.
  4. Get your versions signed for self-distribution.
  5. Add everything to your repo and deploy. Example.
